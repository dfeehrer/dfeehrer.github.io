DingMux

Demuxes using mp4box.js (BSD-3-Clause).

Write all frames to a 2D HTML canvas, much faster than real time.

Captures frames from the canvas one by one and encodes to H264 using webcodecs VideoEncoder.

Then uses mp4-muxer (MIT license) to mux everything back into a proper mp4 container file.
