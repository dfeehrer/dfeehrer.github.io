LetterCraze
Software Engineering term project: Java implementation of word building game.
Project

Working with 4 other students in my Software Engineering course, I designed, implemented and tested a word-building game in Java, as well as a companion level building application.

We analysed the problem from an object oriented perspective, creating use cases and UML diagrams. We then implemented our solution in phases, using git for version control. We used the MVC design pattern and Swing for the GUI.

One of the major features I emplemented was the persistent file storage system to allow for reading and writing levels between the builder and player application using the Jackson 2 library. We tested our application with JUnit, reaching 80% code coverage.
